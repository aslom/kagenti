## How to Run Per-Model Tests

```bash
# Remote (LiteMaaS) — used in CI
openshell-full-test.sh --skip-cluster-create --skip-cluster-destroy

# Local (Ollama)
openshell-full-test.sh --skip-cluster-create --skip-cluster-destroy \
  --llm-provider ollama --llm-models "qwen3-coder:30b,deepseek-r1:32b"

# View metrics
./parse-test-matrix.sh --metrics /tmp/kagenti/tdd-iter8/llm-metrics.json
```
