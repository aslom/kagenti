kagenti-teleport-setup

Self-contained setup script for kosh CLI and OpenShell gateway access. Installs everything needed with no kubectl or cluster access required.

Download kagenti-teleport-setup.py

Quick Start

# Download and run

curl -sSfk -O https://kagenti-teleport-setup-team1.apps.epoc002.ete14.res.ibm.com/kagenti-teleport-setup.py
uv run kagenti-teleport-setup.py --user alice --password alice123 --test

# Direct uv run (requires TMPDIR to exist)
mkdir -p "${TMPDIR:-/tmp}" && \
uv run https://kagenti-teleport-setup-team1.apps.epoc002.ete14.res.ibm.com/kagenti-teleport-setup.py \
  --user alice --password alice123 --test

What it does

  1. Checks for uv, exits with install instructions if missing
  2. Installs kosh.py + support files to $XDG_CONFIG_HOME
  3. Configures openshell gateway (OIDC auth mode)
  4. Installs embedded mTLS certificates (CA + client cert/key)
  5. Performs headless OIDC login via PKCE flow
  6. Tests with kosh sandbox list

Prerequisites
